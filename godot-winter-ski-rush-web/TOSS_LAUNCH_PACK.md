# Winter Ski Rush Toss Launch Pack

Last updated: 2026-03-19

## 1) 토스 제출용 한줄 소개

- 지름길과 장애물을 읽으며 기록을 줄여 나가는 세로형 스키 타임어택 게임

## 2) 토스 제출용 짧은 소개문

- 코스를 따라 내려가며 체크포인트를 지나고, 위험한 지름길을 파고들어 더 빠른 기록에 도전하는 캐주얼 스키 타임어택 게임입니다.

## 3) 토스 제출용 상세 소개문

- `Winter Ski Rush`는 다운힐 코스를 따라 내려가며 더 빠른 완주 기록을 만드는 세로형 타임어택 게임입니다.
- 기본 코스만 따라가도 완주할 수 있지만, 트랙 사이로 숨겨진 지름길을 발견하면 큰 시간 이득을 볼 수 있습니다.
- 대신 지름길일수록 트랙 폭이 좁고 장애물 회피가 어려워져 조향과 브레이크 타이밍이 더 중요해집니다.
- `A / D`, `S`, `Shift`, `Space`, `Q / E` 조합과 모바일 터치 버튼을 함께 지원해 짧은 세션으로 반복 도전하기 좋게 설계했습니다.

## 4) 스토어/심사 포인트 정리

- 장르: 타임어택 / 액션 / 캐주얼 스포츠
- 핵심 재미: 코스 판독, 지름길 탐색, 속도 관리, 충돌 리스크 조절
- 조작 방식: 터치 드래그 또는 좌우 입력, 브레이크, 부스트, 점프, 트릭
- 플레이 흐름: 출발 -> 체크포인트 통과 -> 지름길 선택 -> 완주 기록 갱신
- 세션 길이: 짧은 반복 도전에 적합
- 저장 요소: 최고 기록

## 5) 광고 및 수익화 상태

- 현재 버전은 광고가 붙지 않은 상태로 출시 준비합니다.
- 배너, 전면 광고, 리워드 광고 UI 모두 현재는 노출하지 않습니다.
- 체크포인트 자동 복귀 구조라서 광고를 억지로 넣으면 플레이 흐름을 해칠 수 있어 초기 심사 버전에서는 제외합니다.
- 현재 심사 설명에는 `광고 없음` 또는 `광고 미적용 초기 버전` 기준으로 적는 편이 안전합니다.

## 6) 심사 제출 시 운영 메모

- 세로 화면 기준으로 설계된 게임입니다.
- 토스 앱 Safe Area 를 반영합니다.
- 토스 뒤로가기 이벤트는 즉시 종료가 아니라 종료 확인 모달로 연결됩니다.
- 토스 게임 계정 키를 받을 수 있으면 사용자별 기록 저장 경로를 분리합니다.
- 브라우저 미리보기에서는 동일 구조를 유지하면서 fallback 저장을 사용합니다.
- 앱이 백그라운드로 가면 게임과 사운드는 자동으로 일시정지됩니다.

## 7) 썸네일 문안 제안

### A안

- 메인 타이틀: `Winter Ski Rush`
- 서브 문구: `지름길을 읽고 기록을 줄이세요`

### B안

- 메인 타이틀: `Winter Ski Rush`
- 서브 문구: `세로형 다운힐 타임어택`

### C안

- 메인 타이틀: `Winter Ski Rush`
- 서브 문구: `위험할수록 더 빠른 라인`

## 8) 썸네일 비주얼 가이드

- 화면 중앙에 스키어와 커브 구간을 크게 보여줍니다.
- 메인 코스와 지름길 라인을 서로 다른 색으로 대비시켜 보여주면 핵심 재미가 잘 드러납니다.
- 나무, 바위, 체크포인트 기문을 함께 넣어 액션과 리스크를 직관적으로 전달합니다.
- 눈밭의 차가운 블루 톤 위에 골드 포인트를 얹으면 현재 UI 톤과 일관성이 좋습니다.

## 9) QR 테스트 체크리스트

### 기본 동작

- 앱 진입 시 세로 고정이 적용되는지 확인
- Safe Area 상단/하단 여백이 기기별로 깨지지 않는지 확인
- 첫 진입 후 Godot 캔버스 로딩과 시작 버튼이 정상 동작하는지 확인
- 출발 -> 체크포인트 -> 충돌 -> 복귀 -> 완주 흐름이 끊기지 않는지 확인

### 저장/계정

- 최고 기록이 정상 저장되는지 확인
- 앱 재진입 후 최고 기록이 유지되는지 확인
- 토스 게임 계정 키를 받은 경우 사용자별 기록이 분리되는지 확인
- 게임 계정 키를 못 받은 경우 fallback 저장이 동작하는지 확인

### 사운드/백그라운드

- BGM 토글이 정상 동작하는지 확인
- 효과음 토글이 정상 동작하는지 확인
- 홈 이동 또는 백그라운드 전환 시 자동 일시정지되는지 확인
- 복귀 후 런 진행과 사운드 상태가 자연스럽게 이어지는지 확인

### 네비게이션/모달

- 토스 뒤로가기 시 종료 확인 모달이 뜨는지 확인
- 안내 모달 열기/닫기가 정상 동작하는지 확인
- 종료 모달에서 `계속 플레이` 와 `종료` 가 각각 정상 동작하는지 확인
- 모달이 열린 동안 게임 진행이 멈추는지 확인

### 플레이 감각

- 터치 드래그 조향이 실제 기기에서 과하게 민감하지 않은지 확인
- 하단 모바일 버튼이 iPhone 홈 인디케이터와 겹치지 않는지 확인
- 충돌 복귀 후 입력 상태가 꼬이지 않는지 확인
- 지름길 발견과 완주 기록 갱신이 정상 반영되는지 확인

## 10) 출시 직전 최종 체크

1. 실제 서비스 도메인 기준 CORS 허용 Origin 재확인
2. 정책 링크(`privacy`, `terms`, `contact`) 최종 확인
3. 대표 썸네일 / 소개 문구 / 카테고리 정보 정리
4. QR 테스트 결과 재확인
5. 광고 미적용 상태로 제출 문구 정리
