# 게임 평가표

최종 업데이트: 2026-02-27 (웹 하이퍼캐주얼 데이터 통합)
기준 문서: 앞으로 게임 개선 피드백 답변은 이 파일을 단일 기준으로 사용

## 데이터 통합 기준
- 벤치마크 소스: Poki / itch.io / CrazyGames 상위 웹 하이퍼캐주얼 트렌드 + 플레이어 피드백 + X 관찰 데이터
- 장르 범위: 설치 없는 웹 하이퍼캐주얼(세션 3~5분, 즉시 이해 가능한 조작)
- 성공 핵심: `5초 Hook + 잦은 보상 + 성장 루프 + 리플레이 동기`
- 제외 항목: 성능/FPS/Poki SDK/엔진 기술 요소는 총점에서 제외(참고 지표로만 별도 관리)

## 점수 규칙 (v5.1)
- 총점 범위: `0~100`
- 계산식: `총점 = Σ((원점수 1~10 / 10) * 가중치)` 후 반올림
- 상태 분류:
  - `히트급` : 85+ (Poki Top 100 잠재력, D1 40%+ 목표)
  - `운영` : 70~84 (itch.io 라이브 가능, 평균 세션 5분+ 목표)
  - `개선` : 50~69 (루프/성장 재설계 우선)
  - `제거` : 50 미만 (핵심 재미 재정의)
- 본 문서 점수는 `내부 플레이/코드 구조 기반 사전평가`이며, 10인 베타 후 재보정한다.

## 평가 프로세스 (필수)
1. 10명 베타 테스트: 각 10분 플레이 후 Google Forms 제출
2. 설문 질문: `재미(1~10)`, `다시 플레이 의향`, `지루한 구간`, `신선한 구간`
3. 기록 지표: 평균 세션 시간, 1인당 리플레이 수, 2일 후 재방문율(D1 시뮬)
4. 합격 기준: `재밌다/다시 하겠다` 긍정 응답 70%+, 리플레이 3회+ 목표
5. 재평가: 베타 결과를 아래 원점수 7개 카테고리에 재반영
