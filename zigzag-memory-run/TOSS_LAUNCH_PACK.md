# Zigzag Memory Run Toss Launch Pack

Last updated: 2026-03-19

## 1) 토스 제출용 한줄 소개

- 좌우 패턴을 기억하고 그대로 복기하는 몰입형 메모리 퍼즐 게임

## 2) 토스 제출용 짧은 소개문

- 잠깐 보여주는 지그재그 경로를 기억한 뒤, 같은 순서로 왼쪽과 오른쪽을 입력해 점수를 올리는 기억 퍼즐 게임입니다.

## 3) 토스 제출용 상세 소개문

- `Zigzag Memory Run`은 매 라운드마다 짧게 제시되는 좌우 경로를 기억한 뒤 같은 순서로 복기하는 메모리 퍼즐 게임입니다.
- 초반에는 짧은 패턴으로 시작하지만 연속 성공이 이어질수록 경로가 길어지고 점수 배수도 커집니다.
- 조작은 단순하지만, 미리보기 구간에서 얼마나 빠르게 패턴을 읽고 입력 단계에서 침착하게 복기하느냐가 핵심입니다.
- 화면 왼쪽과 오른쪽 탭, 또는 키보드 방향키로 바로 플레이할 수 있어 모바일에서도 직관적으로 즐길 수 있습니다.

## 4) 스토어/심사 포인트 정리

- 장르: 기억 퍼즐 / 반응 속도 / 캐주얼
- 핵심 재미: 짧은 관찰, 즉시 복기, 연속 성공에 따른 몰입감
- 조작 방식: 화면 좌/우 탭 또는 방향키 좌/우
- 플레이 흐름: 패턴 미리보기 -> 입력 -> 성공 시 다음 라운드 -> 오답 시 종료
- 세션 길이: 짧게 반복 플레이하기 좋은 구조
- 저장 요소: 최고 기록, 사운드 설정

## 5) 광고 및 수익화 상태

- 현재 버전은 광고가 붙지 않은 상태로 출시 준비합니다.
- 배너, 전면 광고, 리워드 광고 UI 모두 현재는 노출하지 않습니다.
- 추후 사업자 등록과 광고 그룹 생성 이후, 게임오버 시점의 보상형 광고를 우선 검토합니다.
- 현재 심사 설명에는 `광고 없음` 또는 `광고 미적용 초기 버전` 기준으로 적는 편이 안전합니다.

## 6) 심사 제출 시 운영 메모

- 세로 화면 기준으로 설계된 게임입니다.
- 토스 앱 Safe Area 를 반영합니다.
- 토스 뒤로가기 이벤트는 즉시 종료가 아니라 종료 확인 모달로 연결됩니다.
- 토스 게임 계정 키를 받을 수 있으면 사용자별 최고 기록을 분리 저장합니다.
- 브라우저 미리보기에서는 로컬 저장소 fallback 을 유지합니다.
- 앱이 백그라운드로 가면 게임과 사운드는 자동으로 일시정지됩니다.

## 7) 썸네일 문안 제안

### A안

- 메인 타이틀: `Zigzag Memory Run`
- 서브 문구: `기억한 경로를 그대로 복기하세요`

### B안

- 메인 타이틀: `Zigzag Memory Run`
- 서브 문구: `좌우 패턴 기억 퍼즐`

### C안

- 메인 타이틀: `Zigzag Memory Run`
- 서브 문구: `짧게 보고, 정확히 입력`

## 8) 썸네일 비주얼 가이드

- 화면 중앙에 지그재그 경로를 크게 보여줍니다.
- 시작점과 목표점을 서로 다른 색으로 강조합니다.
- 왼쪽/오른쪽 입력을 직관적으로 떠올릴 수 있게 화살표 또는 분기 라인을 넣습니다.
- 실제 플레이 화면처럼 어두운 배경 + 민트/골드 포인트 컬러를 유지하면 현재 UI와 일관성이 좋습니다.

## 9) QR 테스트 체크리스트

### 기본 동작

- 앱 진입 시 세로 고정이 적용되는지 확인
- Safe Area 상단/하단 여백이 기기별로 깨지지 않는지 확인
- 첫 진입 후 시작 버튼과 탭 입력이 모두 동작하는지 확인
- 패턴 미리보기 -> 입력 -> 라운드 진행 흐름이 끊기지 않는지 확인

### 저장/계정

- 최고 기록이 정상 저장되는지 확인
- 앱 재진입 후 최고 기록이 유지되는지 확인
- 토스 게임 계정 키를 받은 경우 사용자별 저장이 분리되는지 확인
- 게임 계정 키를 못 받은 경우 fallback 저장이 동작하는지 확인

### 사운드/백그라운드

- BGM 토글이 정상 동작하는지 확인
- 효과음 토글이 정상 동작하는지 확인
- 홈 이동 또는 백그라운드 전환 시 자동 일시정지되는지 확인
- 복귀 후 일시정지 상태와 사운드 상태가 자연스럽게 이어지는지 확인

### 네비게이션/모달

- 토스 뒤로가기 시 종료 확인 모달이 뜨는지 확인
- 안내 모달 열기/닫기가 정상 동작하는지 확인
- 종료 모달에서 `계속 플레이` 와 `종료` 가 각각 정상 동작하는지 확인
- 모달이 열린 동안 게임이 진행되지 않는지 확인

### 플레이 감각

- 좌/우 탭 판정이 화면 절반 기준으로 자연스러운지 확인
- 미리보기 중 탭 난사 시 오입력 처리나 비정상 진행이 없는지 확인
- 연속 성공 시 점수 상승과 패턴 길이 증가가 의도대로 보이는지 확인
- 오답 시 게임오버 모달 수치가 정확한지 확인

## 10) 출시 직전 최종 체크

1. 실제 서비스 도메인 기준 CORS 허용 Origin 재확인
2. 정책 링크(`privacy`, `terms`, `contact`) 최종 확인
3. 대표 썸네일 / 소개 문구 / 카테고리 정보 정리
4. QR 테스트 결과 재확인
5. 광고 미적용 상태로 제출 문구 정리
