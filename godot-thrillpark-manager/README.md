# ThrillPark Manager (Godot)

브라우저용 테마파크 경영 시뮬레이션 MVP.

## 조작
- `Left Click / Tap`: 건설
- `Right Drag / Touch Drag`: 카메라 이동
- `Wheel` 또는 `+ / -`: 줌
- `1..6`: 건설 모드 전환
- `P`: 일시정지
- `F`: 배속 전환
- `S`: 저장
- `L`: 불러오기
