# TapTapCho - BMGD Quick Brief (Sample Run)

## Project Vision
Build a web-first hyper-casual game platform where users can instantly start short sessions (3-5 minutes), then naturally move into repeat play across multiple game genres.

## Target Players
- Mobile web users who prefer instant play without installation
- Casual players with short play windows
- Social/competitive users who respond to rankings and progression

## Core Product Loop
1. Open platform landing page
2. Start game in one tap
3. Get quick reward/score feedback in under 10 seconds
4. Retry or switch to another recommended game
5. Record progress/score and return

## Differentiation Direction
- Fast content iteration with multiple lightweight game modules
- Shared UX language across games (controls, feedback, retention hooks)
- Ranking + challenge loops for replay and social competition

## Immediate Risks
- Inconsistent game quality between titles
- Weak first-session hook in lower-scoring games
- Mobile readability/performance regressions in some UIs

## Next 2 Sprint Focus
- Raise Sunken Sixway Defense to stable high-quality state (target GQS 80)
- Standardize thumbnail, control UX, and feedback polish across top games
- Set a telemetry baseline (session length, retries, stage reach, retention proxy)
- Prepare ad/reward placement candidates for A/B testing

## Success Metrics (Short-Term)
- Median session time >= 5 min (top 5 games)
- Replay count >= 3 per session (top 5 games)
- Mobile completion/clear progression improvement in key titles
- At least 2 games ready for external portal submission-quality polish
