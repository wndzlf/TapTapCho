# 선큰 식스웨이 디펜스 토스 출시 문안

## 앱 이름

선큰 식스웨이 디펜스

## 한 줄 소개

육방향 길목에 선큰을 배치하고 융합해 50 스테이지를 버티는 캐주얼 타워 디펜스 게임

## 짧은 소개

기본 선큰으로 방어선을 세우고, 속사·광역·기절·탱커 계열로 조합을 넓히며, 코어 공방 성장까지 챙겨 황제를 끝까지 지켜내는 토스용 전략 게임입니다.

## 상세 소개

`선큰 식스웨이 디펜스`는 여섯 방향으로 갈라지는 전장 위에 선큰을 배치해 적을 막아내는 실시간 타워 디펜스입니다.  
스테이지를 클리어할 때마다 골드와 코어를 모아 성장하고, 서로 다른 선큰을 융합해 더 강한 전투 구성을 만들 수 있습니다.  
토스 버전에서는 한글 UI를 중심으로 다시 다듬었고, 가로 화면 고정, Safe Area 대응, 종료 확인 모달, 토스 게임 계정 기준 저장 분리, 사운드 토글 유지까지 반영했습니다.

## 핵심 포인트

- 한 손으로도 확인하기 쉬운 한글 중심 HUD와 안내 문구
- 속사, 광역, 노바, 기절, 탱커, 행운 선큰을 조합하는 성장형 디펜스
- 코어 공방 업그레이드와 융합 시스템으로 매 판 전략이 달라지는 구조
- 토스 미니앱 환경에 맞춘 가로 화면, 저장, 종료 흐름 대응

## 검토 포인트

- 폭력성이나 사행성보다 전략과 성장에 초점을 둔 캐주얼 방어 게임
- 광고 시청 강제 없이 플레이 가능한 구조
- 닉네임 기반 랭킹과 로컬 저장을 지원하되, 토스 게임 계정 키가 있으면 계정 단위로 분리 저장
- 앱 종료와 백그라운드 복귀 시 사용자 흐름이 끊기지 않도록 안전하게 처리

## QR 테스트 체크리스트

- 메인 진입 후 즉시 플레이 가능
- 게임 중 배경 복귀 후 다시 돌아와도 비정상 종료가 없음
- 종료 버튼, 뒤로가기, 홈 이벤트에서 UI가 깨지지 않음
- BGM/효과음 토글, 코어 성장, 랭킹 저장이 정상 동작함
- 법률 페이지 링크가 번들 내부에서 열림
