# Berry Bear Bakery Audio Assets

아래 파일명을 넣으면 게임이 자동으로 사용합니다.
파일이 없어도 기본 합성음(fallback tone)으로 정상 동작합니다.

- `berry-bear-bakery-bgm.mp3`
- `berry-bear-bakery-correct.mp3`
- `berry-bear-bakery-wrong.mp3`
- `berry-bear-bakery-finish.mp3`
- `berry-bear-bakery-tap.mp3`
- `berry-bear-bakery-start.mp3` (선택)

권장: 44.1kHz MP3, 효과음은 300ms 내외로 짧게.
