Winter Ski Rush

다운힐 코스를 읽고 지름길을 파고들며 기록을 줄여 나가는 세로형 스키 타임어택입니다.

웹 미리보기
브라우저가 canvas를 지원하지 않습니다.

토스 게임 계정과 세로 화면 설정을 확인하고 있어요.

EXIT CHECK

Winter Ski Rush를 종료할까요?

진행 중인 런은 저장되지 않지만 최고 기록은 유지됩니다. 토스 앱에서는 닫기 버튼으로 바로 종료돼요.

GAME GUIDE

플레이 안내

코스를 따라 내려가며 시간을 줄이는 타임어택 게임입니다. 트랙 안쪽으로 파고들면 지름길을 발견할 수 있지만, 장애물과 코스 이탈에 더 민감해집니다.

좌우 조향은 A / D 또는 화면 터치 드래그, 브레이크는 S, 부스트는 Shift, 점프는 Space, 공중 트릭은 Q / E로 조작합니다. 모바일에서는 하단 터치 버튼도 함께 사용할 수 있어요.

토스 앱이 백그라운드로 전환되거나 종료 확인 모달이 열리면 게임과 사운드는 자동으로 일시정지됩니다.
