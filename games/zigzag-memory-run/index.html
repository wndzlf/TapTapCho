GAME GUIDE

플레이 안내

매 라운드마다 좌우 경로가 잠깐 제시됩니다. 미리보기가 끝나면 같은 순서로 왼쪽 또는 오른쪽을 입력하세요.

연속으로 성공할수록 점수 보너스가 커지고 패턴 길이도 늘어납니다. 토스 앱이 백그라운드로 전환되면 자동으로 일시정지됩니다.
