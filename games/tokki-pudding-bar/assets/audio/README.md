# Tokki Pudding Bar Audio Assets

이 게임은 오디오 파일이 없어도 합성음(fallback tone)으로 동작합니다.

원하면 아래 이름의 파일을 추가할 수 있습니다.

- `tokki-pudding-bar-start.mp3`
- `tokki-pudding-bar-perfect.mp3`
- `tokki-pudding-bar-good.mp3`
- `tokki-pudding-bar-miss.mp3`
- `tokki-pudding-bar-finish.mp3`

권장: 44.1kHz MP3, 효과음은 짧고 가볍게.
