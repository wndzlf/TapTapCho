# Nyang Jelly Cafe Audio Assets

아래 파일명을 그대로 넣으면 게임이 자동으로 적용합니다.
파일이 없으면 기본 합성음(fallback tone)으로 동작합니다.

- `nyang-jelly-cafe-bgm.mp3`
- `nyang-jelly-cafe-serve.mp3`
- `nyang-jelly-cafe-miss.mp3`
- `nyang-jelly-cafe-timeout.mp3`
- `nyang-jelly-cafe-tap.mp3`
- `nyang-jelly-cafe-start.mp3` (선택)

권장: 44.1kHz MP3, 효과음은 300ms 내외로 짧게.
